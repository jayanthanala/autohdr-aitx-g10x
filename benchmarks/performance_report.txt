SPARK STORY
----------------------------------------
Why NVIDIA DGX Spark for AutoHDR?

1. 128GB Unified Memory
   - Entire dataset (577 images) fits in memory
   - No disk I/O bottleneck during training
   - Enables larger batch sizes

2. Mixed Precision (FP16)
   - 2x faster training with AMP
   - Tensor Core utilization
   - No accuracy loss

3. Local Inference
   - Data privacy (photos never leave device)
   - Low latency (~40ms per image)
   - No cloud costs

4. TensorRT Ready
   - FP16 optimization for production
   - Further 2x speedup possible
